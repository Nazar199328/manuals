Установка
---------
$ sudo pacman -S  awesome

Для автоматического конфигурирования меню установить пакеты:
$ yaourt -S awesome-freedesktop-git
$ yaourt -S awesome-scratchpad-git

Горячие клавиши awersome
------------------------

Mod4 + Left/Right - перемещение по тегам (виртуальным рабочим столам)
Mod4 + Esc - вернуться на предыдущий тег
Mod4 + 1, 2, 3 - перемещение по тегам

Mod4 + R - запуск исполняемого файла
Mod4 + P - меню
Left & Right - перемещение по меню
Enter - перейти в пункт меню
Backspace - вернуться назад
Esc - выти из меню

Mod4 + Space - переключение layout'ов "вперед"
Mod4 + Shift + Пробел - переключение layout'ов "назад"
Mod4 + J & Mod4 + K - переключение между окнами (J - перемещение "вперед", K - "назад")
Mod4 + Tab - преключение на предыдущее активное окно
Mod4 + Shift + J & Mod4 + Shift + K - поменять окна местами
Mod4 + ЛКМ - перемещение окна мышью
Mod4 + Shift + Номер тега - перемещение окна в другой тег

Mod4 + H & Mod4 L - изменение размера окна
Mod4 + ПКМ - изменение размера окна
Mod4 + F - полноэкранный режим
Mod4 + Shift + C - закрыть окно
Mod4 + Enter - запуск терминала по умолчанию
Mod4 + W - запуск настраиваемого меню awersome

Mod4 + Ctrl + Space - делает окно "плавающим", не привязанным к layout-у: можно его двигать и менять размер. Чтобы снова зафиксировать его, нажать комбинацию еще раз

Mod4 + Ctrl + Enter - делает окно "главным", первым

Mod4 + Ctrl + R - рестарт оконного менеджера awersome
Mod4 + Shift + Q - выход из оконного менеджера awersome

Mod4 + n - свернуть окно
Mod4 + m - развернуть окно на весь экран

Чтобы приложение запускалось при старте awesome, добавить строчку в .rc.lua:
os.execute("pgrep -u $USER -x xxkb || (xxkb &)")
