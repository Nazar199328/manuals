Горячие клавиши Sublime text
-----------------------------

Работа с файлами
----------------
Ctrl-N - новый документ
Ctrl-O - открыть файл
Ctrl-S - сохранить текущую вкладку
Ctrl-S - вызвать окно сохранения файла
Alt-1,2,3 - перемещение по вкладкам
Ctrl-W - закрыть текущую вкладку
Ctrl-Shirt-T - открыть последний закрытый файл

Работа с текстом
----------------
Home - в начало строки
End - в конец строки
Ctrl-Shift-Left - выделить слово слева
Ctrl-Shift-Right - выделить слово справа
Ctrl-Shift-D - продублировать строку
Ctrl-Shift-K - удалить строку

Ctrl-M - перейти к соответствующей скобке (закрывающейся или открывающейся)
