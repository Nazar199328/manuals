Справка по Blender.
------------------

Операции над окнами/видами.
--------------------

Shift + Пробел - развернуть окно на весь экран
Shift + Scroll (колесо мыши) - передвижение вида вверх/вниз
Ctrl + Scroll - передвижение вида вправо/влево
Scroll - вращение вида, изменение масштаба
Num5 - изометрический вид (убирает перспективу)
T - показывает/скрывает панель Object Tools
N - показывает/скрывает панель Transform (окно вида)
Shift + S - задание положения курсора
File -> Load Factory Setting - восстановление заводских настроек

Чтобы вставить картинку в вид: N -> Background images -> Add Image

Редактирование объектов.
------------------------

Правая кнопка мыши (ПКМ) - выделить объект в объектном режиме, точечное выделение в режиме редактирования
A - выделить все / снять все выделения
Shift  + ПКМ - выделение нескольких объектов
B - прямоугольное выделение
С - выбор объектов группы
Z - режим сетки

Свойства объекта, такие как координаты, угол поворота, масштаб расположены на панели Transform (кнопка N)

Tab - объектный режим / режим редактирования
Ctrl + Tab - режимы редактирования (полигоны, грани, вершины)

Shift + X - удаление объекта
Shift + A - добавление объекта

Shift + D - копия выделенного объекта
G - перемещение объетов
R - вращать объект, затем X, Y, Z, затем угол в градусах
S - масштабировать объекты, затем X, Y, Z для масштабирования по одной из осей
Выделить объект, точка на цифровой клавиатуре - центрирование относительно объекта
Ctrl + Z - отменить действие
Shift + Ctrl + Z - повторить отмененное действие
T -> Transform -> Origin - координирование объекта

E - extrude, экструдирование, выдавливание
Alt + M - объединить в центре
F - создать полигон на выделенных вершинах
Ctrl + R - добавление ребер

Для отображения нормалей в режиме редактирования (Tab) N -> Mesh Display ->Normals: иконка Face
Для измения ориентации нормалей - выделить полигоны, меню Mesh -> Normals -> Fip Normals
Для того, чтобы все нормали смотрели наружу - выделить весь mesh (кнопка А), затем Ctrl + N

Рендеринг.
----------
F12 - рендер
Esc - вернуться в 3D
