Переодеваем напарников:
----------------------
1. Подходим к любому напарнику.
2. Открываем консоль и кликаем на напарника, в центре экрана должно появится ID, у Лидии это 000A2C94
3. Вводим команду showinventory . Появится список предметов в "рюкзаке" напарника. В том числе и "скрытые" предметы.
4. Смотрим на ID этих предметов, например, у Лидии есть лук, который не убрать, его номер 10E2DD
5. В консоли вводим removeitem 10E2DD 1. Всё, лука нет. Если он всё же есть - повторяем до тех пор, пока он не исчезнет.
6. С другими вещами проводим все те же манипуляции.
Небольшая поправка:
Все предметы удалить не получится, надо оставить хотя бы 1 "дефолтную" вещь, в моем случае это сапоги. Если все убрать, то все вернется.

Настройка
---------
Для задания пользователького разрешения экрана в файле SkyrimPrefs.ini задать значения переменных:
isize H=830
isize W=1580

Для корректной настройки отображения текстур в файле SkyrimPrefs.ini
[Display]
iBlurDeferredShadowMask=4
bFloatPointRenderTarget=1
bTreesReceiveShadows=1
bDrawLandShadows=1
bShadowsOnGrass=1

Для того, чтобы вода была прозрачной в файле SkyrimPrefs.ini:
bDoDepthOfField=0
iRadialBlurLevel=0

Конструктор Creation kit.
------------------------
Тело и голова NPC разного цвета.
В редакторе в Object Window откройте вкладку actors, найдите в списке своего персонажа, выделите его (один клик правой клавишей) и нажмите Ctrl+F4. Как правило это помогает.

Чтобы отредактировать нужный esp нужно нажать кнопку Set as Active.

Кого достают ошибки конструктора при загрузке плагинов в файле SkyrimEditor.ini найдите параметр bBlockMessageBoxes и присвойте значение 1

Чтоб редактор видел русский язык в файл Skyrim\SkyrimEditor.ini в раздел [General] добавить первой строкой Slanguage=RUSSIAN

Для  загрузки нескольких мастер-файлов в SkyrimEditor.ini в секцию [General] нужно добавить строку: bAllowMultipleMasterLoads=1

Чтобы отключить проверку правописания, нужно в файле SkyrimEditorPrefs.ini заменить "bSpellCheckEnabled=1" на "bSpellCheckEnabled=0" (без кавычек). Вдруг кому-то поможет

Для рабочего стола xfce  необходимо освободить комбинацию клавиш Ctrl + F4. Для этого перейти Диспетчер настроек → Диспетчер окно → Клавиатура → Рабочее место 4.

Подключение bsa архивов.
------------------------
Если в СК вы видите:
1) фиолетовые объекты - это значит что СК не нашел файла с текстурами
2) вместо объектов красные ромбы с восклицательным знаком - это значит что СК не нашел файла с моделью.

Возможно это следствие того, что файлы текстур/моделей запакованы в архивы *.bsa, а СК не знает об этом.

В SkyrimEditor.ini дописать в разделе [Archive]- Dawnguard.bsa
В 3х местах:
SArchiveList=
SResourceArchiveList=
SResourceArchiveList2=

Мне кажется что SResourceArchiveList2= является продолжение списка SResourceArchiveList=. Видимо есть какое-то ограничение. Поэтому достаточно прописать в одном из этих списков.

Баги
----
Баги с Сераной
Серана не выходит из  грота:
Выходим из грота и вводим в консоли:
prid 02002B74 — выделяем Серану;
moveto player — перемещаем Серану к игроку.

Серана зависла над трупом:
Убиваем Серану, затем выделяем ее и пишем в консоли сначала disable. Она исчезает. Потом enable — Серана появляется.

Серана ползает на корточках:
Пишем в консоли сначала disable. Она исчезает. Потом enable — Серана появляется.


Самые дорогие зелья
-------------------
Корень жарницы + человеческое сердце (= 6014) + лютый гриб = 6865
Пшеница + ползучая лоза + палец великана = 5479
Бородатый мох + медвежьи когти + палец великана = 5431
Крыло синей бабочки + голубой горноцвет + палец великана = 5373
Палец великана + светящийся гриб + ягоды можжевельника = 4085
Бородатый мох + светящийся гриб + светящаяся пыль = 3804
Мора тапинелла + ползучая лоза + чешуйчатка = 3702 (заметим: все компоненты легко доступны путем сбора!)
Крыло лунного мотылька + перья ястреба + прах вампира = 3454
Бородатый мох + глаз саблезуба + медвежьи когти = 3351
Крыло лунного мотылька + медовые соты + прах вампира = 3313
Полезные рецепты:
Двемерское масло + стержневой корень = зелье имеет 4 свойства (2308)
Алый корень Нирна + корень Нирна = зелье имеет 4 свойства (3097)
Соль + чеснок + (ингредиент замедления) = 1135 (2474, большие рога 2830)
Голубой горноцвет + пшеница + хвост злокрыса = 1640
Голубой горноцвет + пшеница + крыло синей бабочки = 2746
Клешня грязевого краба + жареная злокрысья кожа + (ингредиент сопротивления огню) = 976 (1551)
Жир тролля + ухо фалмера + (ингредиент для навыка двуручного оружия) = 1152 (1911)
Соль + ядовитый колокольчик + яйцо коруса = 1912
Зубы ледяного привидения + яйцо коруса + (кровавый венец или белянка) = 1680 (2180 или 2236).
Заметим, что яйцо коруса, кровавый венец, белянка и остальные виды не древесных
грибов, в больших колличествах есть в Провале сквозняков.
Болотный стручок + бесовский гриб + (ингридиент Затяжному урону здоровья
или сопротивления электричеству) = примерно 1932 (2384 или 2364)
Бородатый мох + паслен + крыло монарха = более 2114.
