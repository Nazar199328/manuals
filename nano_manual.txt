Tекстовый редактор nano
-----------------------

$ cp /etc/nanorc ~/.nanorc
Вносим изменения в .nanorc (строчки уже есть, просто раскомментировать)

-----------------------------------------
# Разрешить перенос длинных строк для отображения их без прокрутки
set softwrap
# При  необходимости, наоборот, запретить перенос строк
set nowrap

# Разрешить установку курсора на позицию в тексте с помощью щелчка мышки
set mouse

# Выводить позицию курсора (строка, столбец, символ) в статусе. Без этого выводится только
# по CTRL-C
set const

# Уменьшим вдвое "размер тубуляции"
set tabsize 4

# Подключаем nanorc файлы C++, HTML, bash
include "/usr/share/nano/c.nanorc"
include "/usr/share/nano/html.nanorc"
include "/usr/share/nano/man.nanorc"
include "/usr/share/nano/sh.nanorc"
-------------------------------------------

Проверить, какие есть подсветки: 
$ ls /usr/share/nano/

Опции 
-----

-B - делать бекапы
-u - включить отмену действий 
-w - запретить перенос длинных строчек
-$ - (softwrap) отображать длинные строчки на нескольких строках
Также можно задать все эти опции в конфиге.

Горячие клавиши
-----------------

Навигация
---------
Начало строки — Home (Ctrl-A)
Конец строки — End (Ctrl-E)
На строчку вверх - стрелка вверх (Ctrl-P)
На строчку вниз - стрелка вниз (Ctrl-N)
Страниц вверх — PgUp (Ctrl-Y)
Страница вниз — PgDn (Ctrl-V)

На один символ вперед - Ctrl-F
На один символ назад - Ctrl-B
На слово вперёд — Ctrl-Space
На слово назад — Alt-Space (курсор встаёт в начале слова)

В начало файла — Alt-| или Alt-\ (короче, Alt- и «эта» кнопка)
В конец файла — Alt-/ или Alt-? (аналогично, Alt «эта» кнопка)
До первой пустой строки вниз — Alt-0 (или Alt-))
До первой пустой строки вверх — Alt-9 (или Alt-()


Поиск
-----
Найти слово - Ctrl-W
Повторный поиск (не показывает дималог) - Alt-W
Найти и заменить - Ctrl-| или Ctrl-\ (короче, Ctrl- и «эта» кнопка)

Редактирование
--------------
Удалить символ над курсором - Ctrl-D
Удалить символ слева от кусора - Ctrl-H

Отменить действие - Alt-U (undo)
Вернуть отмененное действие - Alt-E (redo)

Alt-L - включить / выключить перенос строк
Alt-G - переход на строку с указанным номером

Ctrl-6 (Alt-A) - начало выделения
Alt-6 - копировать выделенное в буфер (если ничего не выделено, скопирует в буфер текущую строчку)

Ctrl-K (или F9) - вырезать выделение (оно в буфере cutbuffer). При отсутствии выделения вырезает всю строку.

Ctrl-U - вставить содержимое cutbuffer'а

Alt-K - переключает режимы работы комбинации Ctr-K. 
Либо вырезать всю строку, либо от курсора до конца строки.

Alt-T - стирает всё от курсора и до конца файла (команда trunkate).
Ctrl-J - Выровнять по ширине текущий параграф (в котором находится курсор)

Многобуферность
---------------
Включается командой Alt-F. Окна переключается клавишами Alt-< и Alt->.
Если многобуферность активирована, комбинация Ctrl-R откроет файл, путь к которому будет прописан. Если, находясь в диалоге, нажать Ctl-T, то откроется меню выбора файла. Если нет, то Ctrl-R вставляет содержимое указанного файла в открытый файл
