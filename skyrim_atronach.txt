Соли

    Соль + Аметист + Камень душ (любой) = Соль пустоты
    Соль + Рубин + Камень душ (любой) = Огненная соль
    Соль + Сапфир + Камень душ (любой) = Морозная соль 

Существа

    Огненная соль + Рубин = Огненный атронах
    Морозная соль + Сапфир = Ледяной атронах
    Соль пустоты + Аметист = Грозовой атронах
    Череп + Сердце даэдра + (2 куска мяса) = Дремора (необходим Сигильский камень) 

Посохи

    Посох грозового атронаха = Метла + Соль пустоты + Большой камень душ + Орихалковый слиток
    Посох морозного атронаха = Метла + Морозная соль + Большой камень душ + Очищенный лунный камень
    Посох огненного атронаха = Метла + Огненная соль + Большой или Великий камень душ + Орихалковый слиток, либо Корундовый слиток 

Книги заклинаний

    Том заклинаний: Вызов ледяного атронаха = Морозная мириам + Морозная соль + Шкура снежного волка + Испорченная книга
    Том заклинаний: Вызов грозового атронаха = Соль пустоты+ Ядовитый колокольчик + Бивень мамонта + Испорченная книга
    Том заклинаний: Вызов огненного атронаха = Огненная соль + Язык дракона + Шкура медведя (любого) + Испорченная книга
    Том заклинаний: Захват душ = Торакс светлячка + Соль + Камень душ (любой заполненный) + Испорченная книга 
	
Свитки

    Вызов огненного атронаха (Scroll of Conjure Flame Atronach) — Огненная соль + Рулон бумаги + Уголь.
    Вызов ледяного атронаха (Scroll of Conjure Frost Atronach) — Морозная соль + Рулон бумаги + Уголь.
    Вызов грозового атронаха (Scroll of Conjure Storm Atronach) — Соль пустоты + Рулон бумаги + Уголь. 

Даэдрическое оружие и доспехи

    Любое даэдрическое оружие/доспехи (требуется Сигильский камень) = Сердце даэдра + Сердечник генератора центуриона + Черный камень душ + 1 аналогичное эбонитовое оружие/доспехи (НЕ зачарованное).
    Случайная зачарованная даэдрическая броня = Соль пустоты + Эбонитовый слиток + Сердце даэдра + заполненный Великий камень душ.
    Случайное зачарованное даэдрическое оружие = Серебряный меч + Эбонитовый слиток + Сердце даэдра + Заполненный Черный, Большой или Великий камень душ. 

Особое	

    Сердце даэдра = Человеческое сердце + Черный камень душ (любой) (Требуется Сигильский Камень)
    Эликсир +100 % усиления Колдовства = Эктоплазма + Камень душ (любой) + пустая бутылка из-под вина
