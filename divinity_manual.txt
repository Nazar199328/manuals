Мануал по Divinity:Original Sin
--------------------------------

Раскачка:
---------
1. Маг
   Способности: вода 5, воздух 4, чародейство 4, сила воли 3, крепость тела 3;
   Таланты: Всезнайка, Длинные руки, Само пройдет, Стеклянная пушка
   	    (тогда стоит взять щит и качнуть блокирование), Громоотвод (маг воздуха)
2. Стрелок
   Способности: стрельба 5, сила воли 3, крепость тела 3;
   Таланты: Грубиян, Друг животных (берет один из лидеров партии), Само пройдет,
   	    Чутье (можно взять)
3. Авантюрист: путь ассасина 4, сила воли 3, крепость тела 3;
   Таланты: Грубиян, Друг животных, Коварная рука, Само пройдет, Чутье (можно взять),
   	    Абсолютная меткость (для использующего гранаты), Стеклянная пушка (щит),
	    Вечное дитя (если есть стекл. пушка), Рогатка (для использующего гранаты),
4. Боец: военное дело 5, сила воли 3, крепость тела 3;
   Таланты: Грубиян (замедление делает стрела и маг воды), Друг животных,
   	    Само пройдет, Рогатка
   
Крафт:
------
Гранаты:
    Бутылка с маслом + запал = Граната огненного смерча

Стрелы:
    Обычный наконечник + Бутылка с маслом = Замедляющий наконечник
    Обычный наконечник + Бочка со слизью = Отравленный наконечник    
    Олений рог + Нож = Наконечник для сбивающей с ног стрелы
    Зуб + Нож = Оглушающий наконечник
    Зелье сопротивления воздуху + Обычный наконечник = Статический наконечник
    Зелье сопротивления огню + Обычный Наконечник = Разрывной наконечник

Зелья:

Амулеты и прочее:
    Веревка + Волшебный череп = Пояс из черепа (интеллект +1)
    Нитка + Волшебный череп = Амулет из черепа (интеллект +1)
    Набор ювелира + Волшебный череп = Кольцо из черепа (интеллект +1)
    Нитка + Волшебная кроличья лапа = Амулет с кроличьей лапой (скорость +1)
    Нитка + Волшебное перо = Амулет с пером (ловкость +1)
    Веревка + Волшебное перо = Пояс с перьями (ловкость +1, инициатива +1)
Пища:
    Нож + Картофель = Нарубленная картошка	
    Печь + Нарубленный картофель = Жаренный картофель
    Печь + Тесто для рыбного пирога = Пирог из рыбы

Доспехи:
    Обрывки ткани + Иголка с ниткой = Тканевые доспехи
    Обрывки ткани + Волшебные иголка с ниткой = Балахон
    Обрывки ткани + Наковальня = Тканевые сапоги
    Куски кожи + Иголка с ниткой = Кожаные доспехи
    Куски кожи + Наковальня = Кожаные сапоги
    Металлолом + Молот = Металлические доспехи
    Металл. осколки + Наковальня = Металлические сапоги
Усиление доспехов:
    Любые кожаные доспехи + Полевая кухня = +Защита
    Любые металл. доспехи + Наковальня = +Перемещение
    Любые металл. доспехи + Обломки = +Защита
    Любые тканые доспехи + Куски кожи = +Защита
    Любые тканые доспехи + Металлолом = +Защита
