Клавиатурные сокращения MC
Чтобы выжать из Midnight Commander всё, стоит запомнить хотя бы несколько клавиатурных сокращений, которые могут сильно облегчить выполнение задач. 
Клавиатурные комбинации в Midnight Commander имеют особенность: комбинация Ctrl + x c означает, что нужно нажать CTRL и X, отпустить и потом нажать клавишу C.

Итак, некоторые комбинации клавиш Midnight Commander:

Ctrl + l		перерисовать экран
Ctrl + R 		обновляет активную панель;
Ctrl + O		скрыть панели;
Ctrl + X C 		посмотреть/поменять права (chmod) для отмеченных файлов;
Ctrl + X O 		посмотреть/поменять владельца (chown) для отмеченных файлов;
Ctrl + S или Alt + S 	открывает диалог быстрого поиска (вводимые символы отображаются не в командной строке, а в строке поиска);
Ctrl + X A 		открывает диалог виртуальных файловых систем;
Ctrl+ \ 		вызывается меню часто используемых каталогов.
Ctrl + Space		показать размер папки
Ctrl + X J 		показываются все фоновые задачи.
Ctrl + X T 		послать выделенные объекты в командную строку
Ctrl+ENTER (ESC, ENTER)	послать текущий объект в командную строку
Ctrl + X S 		создать символическую ссылку (symlink)
Ctrl + X Q 		быстрый просмотр содержимого файла на второй панели.
Ctrl + X I 		быстрый просмотр информации о текущем объекте на второй панели.

ESC + T или ALT + T 	cмена вида панели (стандартный, укороченный, расширенный);
ESC + H или ALT + H 	вызов списка последних выполненных команд;
ESC + A			Послать в ком.строку текущий каталог
ESC, O			Открыть на противоположной панели текущий каталог (подсвеченный)(ESC нажать однократно, затем нажать O)
		
Alt + G 		отмечает первый файл или каталог в отображаемом списке на панели
Alt + R 		отмечает средний файл или каталог в отображаемом списке на панели
Alt + J 		отмечает последний файл или каталог в отображаемом списке на панели
ALT + I 		открывает каталог, который находится на текущей панели
Alt + O 		неактивная панель переходит в каталог на который указывает активная панель.
Alt + . 		не показывать или показать скрытые файлы или каталоги.
Alt + , 		переключить режим отображения панелей (вертикально/горизонтально).
Alt + p (ESC + p) 	вернуть предыдущую (предыдущие) командную строку (строки)
Alt + n (ESC+n)		вернуть последующую (последующие) командную строку (строки)
+ (Plus) 		вызывает окно для выделения группы файлов (можно использовать регулярные выражения)
\ (Backslash) 		снимает отметку с группы файлов (действие, обратное тому, которое вызывается по клавише "+").
Alt + TAB (ESC+TAB) 	Крутейшая кнопка для поиска загружаемых файлов (например вы забыли как полностью пишется netcfg. Достаточно набрать в командной строке net и нажать Alt-TAB - вам сразу покажут что можно запустить, которое начинается со слова net. Нажимать два раза!
Alt + TAB (ESC+TAB)	работает так же как в Bash TAB (т.е. подгоняет по первым буквам имена файлов или директорий. Наберите cd /u потом Alt-TAB - сразу будет cd /usr/ и т.д.)
Alt + Shift + ? 	Диалог поиска файлов

Настройки MC
-------------
Скины лежат здесь: /usr/share/mc/skins
Чтобы запустить скин, например xoria256: 
$ mc -S xoria256
Или найти в конфиге ~/.config/mc/ini переменную skin и заменить default на новое значение:
skin=xoria256
Для прозрачного фона присвоить значение base_color:
base_color=normal=white,default

*Меняем редактор и просмотрщик на встроенные (mcedit) - F9->настройки>-конфигурация. Отмечаем Встроенный редактор и Встроенный просмотр

*В меню терминала Правка->Комбинации клавиш. Переназначить меню и справку, чтобы освободить клавиши F1 и F10


*Удобно иногда открыть текущий каталог в Графическом файловом менеджере - добавим пункт в меню пользователя (F2)
F9->команды->файл меню. Выбрать Пользовательский
O       Открыть Folder
	nautilus "%d"

*Открытие изображений в Eye of Gnome
F9->команда->файл расширениий. Файле пишем:
include/image
   Open=(eog %f &)
   #Open=(see %f &)
   #View=%view{ascii} identify %f
   View=%view{ascii} asciiview %f

*Открытие видеофайлов в vls
include/video
	#Open=(mplayer %f >/dev/null 2>&1 &)
	#Open=(gtv %f >/dev/null 2>&1 &)
	#Open=(xanim %f >/dev/null 2>&1 &)
	Open=(vlc %f >/dev/null 2>&1 &)

Копирование в буфер:
Ctrl + F - копирование в буфер
Shift + F5 - вставка из буфер
Если не получается, проверяем владельца  каталога mcedit (ставим пользователя)

Пользовательское меню
---------------------
Настраиваетсяв файле mc/menu
Макроподстановки:
%f имя текущего файла.
%d имя текущего каталога.
%F имя текущего файла в неактивной панели.
%D имя текущего каталога неактивной панели.
%t помеченные файлы.
%T помеченные файлы из неактивной панели.
